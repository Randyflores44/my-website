Aquassay Project

Project information

  • Category: SCADA
  • Client: Nestle Waters
  • Project date: 20 November, 2020

‘Aquassay’ SCADA Project for water data analysis for the whole factory.

  • Used 14 Siemens PLCs Connected via Ethernet Network.
  • Created control cabinet, installed, wired.
  • Written every PLC program from scratch.
  • Used AVEVA ‘InTouch’ software as OPC UA Server, configured the communications parameters and tags.
  • Used Custom Web application (DMCG Data Pusher), to push data to the ‘Aquassay’ website.
